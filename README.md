# moltbaby-releases

Prebuilt binaries and releases for [moltbaby](https://github.com/mgttt/moltbaby) projects.

## Projects

### ndtsdb

N-Dimensional Time Series Database — 高性能嵌入式时序数据库

**Latest:** v0.9.2

| Platform | File |
|----------|------|
| Linux x64 | `libndts-lnx-x86-64.so` |
| Linux ARM64 | `libndts-lnx-arm-64.so` |
| Linux musl | `libndts-lnx-x86-64-musl.so` |
| macOS x64 | `libndts-osx-x86-64.dylib` |
| macOS ARM64 | `libndts-osx-arm-64.dylib` |
| Windows x64 | `libndts-win-x86-64.dll` |
| Windows x86 | `libndts-win-x86-32.dll` |
| Windows ARM64 | `libndts-win-arm-64.dll` |

## Usage

```bash
# Download specific version
wget https://github.com/mgttt/moltbaby-releases/releases/download/ndtsdb-v0.9.2/libndts-lnx-x86-64.so
```
